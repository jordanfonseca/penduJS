# penduJS
`Angular 1.5 ES6 Game`

# Clone the repository
git clone https://github.com/jordanfonseca/penduJS.git
# Install dependencies
npm install
# Development mode
npm run dev
# Production mode
npm run prod
# Start the server
npm run start
# Go to
http://127.0.0.1:8080/
